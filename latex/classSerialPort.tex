\hyperlink{classSerialPort}{Serial\-Port}

Class to manage the communication with a serial port.

R\-S 22-\/812 sends 9bytes long packets with the codified information. This class is meant to read those packets and send it to the \hyperlink{classRS22812}{R\-S22812} class to store and interpret.
